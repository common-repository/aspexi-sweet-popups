== Description ==

Simple popups plugin based on Sweet Alert that automatically centers itself on the page and looks great no matter if you're using a desktop computer, mobile or tablet.

####Free version####

* Automatically open popup window on any page or front page only
* Beautiful [Sweet Alert](http://t4t5.github.io/sweetalert/) popups
* Custom popup header and content (simple HTML and shortcodes)
* Show only once per user (based on cookies)
* Few types of predefined icons
* Themes (Default, Facebook, Google, Twitter)
* Good-looking popup on desktop, mobile or tablet

== Installation ==

1. Upload `aspexi-sweet-popups` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin (Appearance -> Aspexi Sweet Popups)
